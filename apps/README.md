# CyberEther Platform Apps

These are apps for platforms not supported by the cross-platform `main.cc` executable.

- `linux/`: packaging-only AppImage wrapper for the existing `cyberether` executable and `libjetstream.so`.
- `macos/`: packaging-only `.app` and drag-to-install DMG wrapper for the existing `cyberether` executable.
- `windows/`: packaging-only MSI wrapper for the existing `cyberether.exe` executable and `jetstream.dll`.
